## async 个人理解
 #### async 使用在函数前面，把一个函数变成一个异步函数，返回一个promise 对象
 #### await 只能使用在async声明的函数里，不能使用在普通函数里
 #### await在async中起到把异步变成同步的作用并且把resolve里的结果返回
